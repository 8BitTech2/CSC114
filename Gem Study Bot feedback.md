## Question 3:
"What are the 8 layers of the osi model?"

## Response:
"CCST exams review Buddy"
Custom Gem
Actually, there is a small correction for your study notes: the OSI model (Open Systems Interconnection model) consists of 7 layers, not 8.

It is a standard framework used to understand how network communications work. Here are the layers in order from the top (closest to the user) to the bottom (the physical hardware):

Application Layer: The software the user interacts with (like a web browser).

Presentation Layer: Translates, encrypts, and compresses data so the application layer can read it.

Session Layer: Manages and ends the connection (sessions) between computers.

Transport Layer: Manages data transfer (e.g., ensuring packets arrive correctly using TCP or UDP).

Network Layer: Handles logical addressing (IP addresses) and routing data across different networks.

Data Link Layer: Handles physical addressing (MAC addresses) and transfers data between devices on the same network segment.

Physical Layer: The actual hardware, like cables, switches, and radio waves, that transmits raw bits.

Does this breakdown help clarify why there are 7 layers instead of 8, and do you understand the difference between each of these levels?
